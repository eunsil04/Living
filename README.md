# 세종시 데이터 대시보드

세종시의 인구, 교통, 카드매출 데이터를 시각화하는 웹 애플리케이션입니다.

## 기능

- **인구 현황**: 세종시 인구 추이 및 연령대별 분포 시각화
- **교통 현황**: 공공자전거 대여소 및 BRT 노선 현황
- **카드매출 현황**: 행정동별 카드소비 현황 및 월별 추이

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 기술 스택

- React 18
- TypeScript
- Vite
- Recharts (차트 라이브러리)
- PapaParse (CSV 파싱)

## 데이터 파일

프로젝트 루트의 CSV 파일들을 사용합니다:

- `202501_202506_연령별인구현황_월간.csv`
- `202507_202512_연령별인구현황_월간.csv`
- `세종도시교통공사_공공자전거(어울링)_대여소현황_20250106.csv`
- `세종특별자치시_BRT노선별 정류장 현황_20250717.csv`
- `세종특별자치시_카드매출_행정동별 카드소비 현황_20250531.csv`
